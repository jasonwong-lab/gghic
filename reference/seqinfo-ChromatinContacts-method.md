# Get sequence information

Retrieves chromosome/sequence information.

## Usage

``` r
# S4 method for class 'ChromatinContacts'
seqinfo(x)
```

## Arguments

- x:

  ChromatinContacts object.

## Value

Seqinfo object.

## Examples

``` r
if (FALSE) { # \dontrun{
cc <- ChromatinContacts("sample.cool")
seqinfo(cc)
} # }
```
